# Java Coding Challenge #5

**Topic:** Sliding Window
**Difficulty:** Medium

## Problem
Given a string s, return length of the longest substring without repeating characters.

## Constraints
- 0 <= s.length <= 100000

## Example
Input: abcabcbb
Output: 3
Explanation: abc is the longest substring without duplicates.

Use an optimal Java solution with clean naming and clear logic.
