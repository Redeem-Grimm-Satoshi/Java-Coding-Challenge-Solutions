# Java Coding Challenge #3

**Topic:** Hashing
**Difficulty:** Easy

## Problem
Given an integer array and target, return indices of two numbers that add up to target.

## Constraints
- 2 <= n <= 100000

## Example
Input: nums=[2,7,11,15], target=9
Output: [0,1]
Explanation: nums[0] + nums[1] = 9.

Use an optimal Java solution with clean naming and clear logic.
