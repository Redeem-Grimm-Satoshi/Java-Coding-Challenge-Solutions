# Java Coding Challenge #10

**Topic:** Dynamic Programming
**Difficulty:** Hard

## Problem
Given coins and amount, return minimum number of coins to make amount; return -1 if impossible.

## Constraints
- 1 <= coins.length <= 100
- 0 <= amount <= 10000

## Example
Input: coins=[1,2,5], amount=11
Output: 3
Explanation: 11 = 5 + 5 + 1.

## Approach
Use an optimal Java solution with clean naming and clear logic.
