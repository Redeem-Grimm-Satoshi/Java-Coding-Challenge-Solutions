# Java Coding Challenge #9

**Topic:** Binary Search
**Difficulty:** Medium

## Problem
Given a sorted array and target, return index if found else -1.

## Constraints
- 1 <= n <= 1000000

## Example
Input: nums=[-1,0,3,5,9,12], target=9
Output: 4
Explanation: 9 exists at index 4.

## Approach
Use an optimal Java solution with clean naming and clear logic.
