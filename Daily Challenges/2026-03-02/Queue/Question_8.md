# Java Coding Challenge #8

**Topic:** Queue
**Difficulty:** Medium

## Problem
Implement a moving average over a data stream with fixed window size k.

## Constraints
- 1 <= k <= 10000

## Example
Input: k=3, stream=[1,10,3,5]
Output: [1.0, 5.5, 4.67, 6.0]
Explanation: Average of the last k elements at each step.

## Approach
Use an optimal Java solution with clean naming and clear logic.
