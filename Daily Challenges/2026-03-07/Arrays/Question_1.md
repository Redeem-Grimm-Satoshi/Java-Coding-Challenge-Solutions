# Java Coding Challenge #1

**Topic:** Arrays
**Difficulty:** Easy

## Problem
Given an integer array, return the maximum subarray sum using Kadane's algorithm.

## Constraints
- 1 <= n <= 100000
- -10^9 <= arr[i] <= 10^9

## Example
Input: [-2,1,-3,4,-1,2,1,-5,4]
Output: 6
Explanation: The subarray [4,-1,2,1] has sum 6.

## Approach
Use an optimal Java solution with clean naming and clear logic.
